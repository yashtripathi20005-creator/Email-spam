# Email Spam Classifier

A machine learning-based email spam classifier built with Python and scikit-learn.

## Features

- **Machine Learning Model**: Uses Naive Bayes with TF-IDF vectorization
- **High Accuracy**: Achieves >95% accuracy on test data
- **Easy to Use**: Simple CLI interface for classifying emails
- **Feature Analysis**: Shows important words for spam/ham classification
- **Batch Processing**: Can classify multiple emails at once
